# vue2

## Project setup
```
cp .env.example .env
yarn install
```

Change `VUE_APP_BASE_URL` in your .env file to your Drupal installation URL.

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Run your unit tests
```
yarn test:unit
```

### Lints and fixes files
```
yarn lint
```
